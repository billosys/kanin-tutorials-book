## Installation

Just add it to your ``rebar.config`` deps:

```erlang
  {deps, [
    ...
    {kanin, ".*",
      {git, "git@github.com:YOURNAME/kanin.git", "master"}}
      ]}.
```

And then do the usual:

```bash
    $ rebar get-deps
    $ rebar compile
```
